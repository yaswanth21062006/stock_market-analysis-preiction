## Stock Market Analysis and Price Prediction

## Project Overview

This project focuses on analyzing and predicting stock prices of selected companies from the Indian stock market using Python. Historical stock data is collected, visualized, and analyzed to understand trends and patterns. A machine learning model is then used to predict stock prices.

## Companies Selected

The following five companies were selected for analysis:

* Asian Paints
* Hindustan Copper
* Kotak Mahindra Bank
* State Bank of India (SBI)
* Tata Consumer

## Tools and Technologies Used

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Data Collection

Historical stock price data from **2015 to 2024** was collected for all five companies. The data includes stock closing prices for each year.

## Data Analysis

The collected data was cleaned and organized using Pandas. Line graphs were created using Matplotlib to visualize the stock price trends of each company across the years.

### Observations

* Asian Paints showed consistent growth over the years.
* Hindustan Copper showed high volatility in stock prices.
* Banking stocks like Kotak Mahindra Bank and SBI showed moderate growth trends.

## Machine Learning Model

A **Linear Regression model** was used to predict future stock prices based on historical data.

### Model Performance

The model achieved an accuracy of approximately **0.9**, indicating a strong relationship between the variables used for prediction.

## Conclusion

This project demonstrates how data analysis and machine learning can be used to understand stock market trends and make predictions based on historical data.
